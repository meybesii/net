Элементы HTML

Элементы HTML состоят из тегов, которые могут быть открывающими и закрывающими. Например, тег <p> используется для определения абзаца.

    Некоторые распространенные элементы HTML включают:

    1. <h1> - заголовок первого уровня
    2. <a> - ссылка
    3. <img> - изображение
    4. <div> - контейнер для других элементов
